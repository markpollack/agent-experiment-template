## Structure

Add domain-specific knowledge files to this directory. Each file should contain
focused, actionable knowledge that helps the agent perform its task better.

Knowledge files are the **independent variable** in ablation experiments — adding
or removing knowledge files between variants is how you measure knowledge impact.
