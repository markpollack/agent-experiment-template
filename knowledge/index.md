```
knowledge/
├── index.md          ← You are here (routing table)

```

Knowledge files are the **independent variable** in ablation experiments — adding
or removing knowledge files between variants is how you measure knowledge impact.

Each file should contain focused, actionable knowledge: patterns to follow,
anti-patterns to avoid, and domain-specific decision criteria.
